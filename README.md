
# Mario on Vim

![](https://raw.github.com/rbtnn/mario.vim/master/mario.gif)


## How to use

        " Start mario.
        :Mario


## Operation Method

* <Space> : Jump mario.
* h : Move mario to the left.
* l : Move mario to the right.
* q : exit mario.vim(same as GameOver).


## Dependency plugin

* [rbtnn/game\_engine.vim](https://github.com/rbtnn/game_engine.vim)
